Linux ејҖеҸ‘зі»еҲ—з¬”и®°(1.3) - йқҷжҖҒеә“дёҺеҠЁжҖҒеә“еҲ¶дҪң

еҸ‘иЎЁдәҺ2023-08-24|жӣҙж–°дәҺ2023-08-24|LinuxLinux ејҖеҸ‘зі»еҲ—з¬”и®°(зі»еҲ—)
|еӯ—ж•°жҖ»и®Ў:1k|йҳ…иҜ»ж—¶й•ҝ:3еҲҶй’ҹ|йҳ…иҜ»йҮҸ:

е»әз«ҷжҸҗдәӨеҺҶеҸІж–Үз« пјҢеҺҹж–ҮеҶҷдҪңж—¶й—ҙ 2023 е№ҙ 2 жңҲеүҚеҗҺгҖӮ

еӣһеҲ°ж–Үз« зӣ®еҪ•

йқҷжҖҒеә“дёҺеҠЁжҖҒеә“еҲ¶дҪң

д»Ӣз»Қ

  • еә“ж–Үд»¶жҳҜи®Ўз®—жңәдёҠзҡ„дёҖзұ»ж–Үд»¶пјҢеҸҜд»ҘжҠҠе®ғеҪ“дҪңжҳҜдёҖдёӘд»Јз Ғд»“еә“пјҢеҸҜд»ҘдёәзЁӢеәҸжҸҗдҫӣеҸҳйҮҸгҖҒеҮҪж•°е’Ңзұ»гҖӮ

  • еә“ж–Үд»¶дёҺеҸҜжү§иЎҢж–Үд»¶е·®еҲ«дёҚеӨ§пјҢдҪҶжҳҜеә“ж–Үд»¶дёҚиғҪзӣҙжҺҘиҝҗиЎҢгҖӮ

  • еә“еҲҶдёәдёӨз§ҚпјҡйқҷжҖҒеә“гҖҒеҠЁжҖҒеә“пјҲе…ұдә«еә“пјүпјҢйқҷжҖҒеә“еңЁзј–иҜ‘ж—¶еҶҷе…ҘеҸҜжү§иЎҢж–Үд»¶пјҢеҠЁжҖҒеә“еңЁжү§иЎҢж—¶иҜ»е…ҘеҶ…еӯҳгҖӮ

  • еә“е…·жңүд»Јз ҒдҝқеҜҶпјҢйғЁзҪІдёҺеҲҶеҸ‘ж–№дҫҝзҡ„дјҳеҠҝгҖӮ

йқҷжҖҒеә“

  • е‘ҪеҗҚи§„иҢғпјҡLinuxпјҡlibxxx.aпјҢWindowsпјҡlibxxx.lib

йқҷжҖҒеә“еҲ¶дҪң

вҖӢ йҰ–е…ҲиҮӘиЎҢе®үиЈ…arе·Ҙе…·гҖӮ

вҖӢ еҒҮи®ҫжҲ‘д»¬зҡ„йЎ№зӣ®еҢ…жӢ¬пјҡmain.cпјҢhead.hпјҢadd.cпјҢsub.cпјҢmul.cпјҢdiv.cгҖӮ

вҖӢ head.cж–Үд»¶дҪҚдәҺincludeзӣ®еҪ•дёӢпјҢadd.cпјҢsub.cпјҢmul.cпјҢdiv.cдҪҚдәҺlibзӣ®еҪ•дёӢгҖӮ

йқҷжҖҒеә“дјҳзјәзӮ№

  • дјҳзӮ№

    • йқҷжҖҒеә“жү“еҢ…еңЁеә”з”ЁзЁӢеәҸдёӯпјҢеҠ иҪҪйҖҹеәҰеҝ«гҖӮ

    • йқҷжҖҒеә“жҸҗдҫӣеҚ•дёҖж–Үд»¶пјҢз§»жӨҚж–№дҫҝгҖӮ

    • йҖӮз”ЁдәҺе°ҸдҪ“з§Ҝеә“гҖӮ

  • зјәзӮ№

    • ж¶ҲиҖ—зі»з»ҹиө„жәҗпјҢйҮҚеӨҚеҚ з”ЁеҶ…еӯҳгҖӮ
    • жӣҙж–°гҖҒйғЁзҪІйә»зғҰгҖӮ

еҠЁжҖҒеә“

  • е‘ҪеҗҚи§„иҢғпјҡLinuxпјҡlibxxx.soпјҢWindowsпјҡlibxxx.dll

еҠЁжҖҒеә“еҲ¶дҪң

жіЁж„ҸпјҡжӯӨж—¶еҗҜеҠЁзЁӢеәҸ./testе°ҶеҮәзҺ°file not foundй”ҷиҜҜпјҢеӣ дёәеҠЁжҖҒеә“и·Ҝеҫ„жІЎжңүиў«жүҫеҲ°гҖӮ

еҠЁжҖҒеә“и·Ҝеҫ„жҗңзҙўеҺҹзҗҶ

  • еҠЁжҖҒеә“еңЁзј–иҜ‘йҳ¶ж®өд»…еҶҷе…ҘжҸҸиҝ°дҝЎжҒҜпјҢеңЁзЁӢеәҸиҝҗиЎҢж—¶иҪҪе…ҘеҶ…еӯҳгҖӮ

  • еҪ“зі»з»ҹйңҖиҰҒиҪҪе…ҘеҠЁжҖҒеә“ж—¶пјҢйҖҡиҝҮеә“еҗҚдёҺз»қеҜ№и·Ҝеҫ„е®ҡдҪҚеҠЁжҖҒеә“пјҢз»қеҜ№и·Ҝеҫ„з”ұеҠЁжҖҒиҪҪе…ҘеҷЁжҹҘжүҫгҖӮ

  • еҜ№дәҺelfж јејҸзҡ„еә”з”ЁзЁӢеәҸпјҢжҹҘжүҫдјҳе…Ҳзә§дёәпјҡ

    DT_RPATHж®ө вҶ’\toвҶ’ зҺҜеўғеҸҳйҮҸLD_LIBRARY_PATH вҶ’\toвҶ’ /etc/ld.so.cacheж–Үд»¶ вҶ’\toвҶ’ /libдёҺ/usr/libзӣ®еҪ•гҖӮ

ж·»еҠ еҠЁжҖҒеә“и·Ҝеҫ„

  • ж–№жі•дёҖпјҡй…ҚзҪ®зҺҜеўғеҸҳйҮҸ

    жіЁпјҡLD_LIBRARY_PATHи·ҹйҡҸз»Ҳз«ҜпјҢйҮҚеҗҜз»Ҳз«Ҝе°ҶдјҡйҮҚзҪ®гҖӮ

  • ж–№жі•дәҢпјҡй…ҚзҪ®/etc/ld.so.cacheж–Үд»¶

    жіЁпјҡ/etc/ld.so.cacheдёәдәҢиҝӣеҲ¶ж–Үд»¶пјҢж— жі•зӣҙжҺҘдҝ®ж”№гҖӮ

  • ж–№жі•дёүпјҡй…ҚзҪ®~/.bashrcж–Үд»¶пјҲдёҚжҺЁиҚҗпјү

    жіЁпјҡ~/.bashrcдёәз»Ҳз«ҜеҗҜеҠЁй…ҚзҪ®пјҢеӣ жӯӨй…ҚзҪ®з»“жқҹйңҖйҮҚеҗҜз»Ҳз«ҜгҖӮ

  • ж–№жі•дёүпјҡж·»еҠ иҮі/libжҲ–/usr/libзӣ®еҪ•пјҲдёҚжҺЁиҚҗпјү

    жіЁж„Ҹпјҡе®№жҳ“ж„ҸеӨ–иҰҶзӣ–йҮҚиҰҒж–Үд»¶

еҠЁжҖҒеә“дјҳзјәзӮ№

  • дјҳзӮ№
    • е®һзҺ°зі»з»ҹиө„жәҗе…ұдә«гҖӮ
    • жӣҙж–°гҖҒйғЁзҪІж–№дҫҝгҖӮ
    • еҠЁжҖҒеә“еҠ иҪҪж—¶еҲ»еҸҜжҺ§еҲ¶гҖӮ
    • йҖӮз”ЁдәҺеӨ§дҪ“з§Ҝеә“гҖӮ
  • зјәзӮ№
    • еҠ иҪҪйҖҹеәҰзӣёжҜ”дәҺйқҷжҖҒеә“иҫғж…ўгҖӮ
    • еҸ‘еёғзЁӢеәҸйңҖжҸҗдҫӣзӣёе…іеә“гҖӮ

C2-Dual (Hard Version) Codeforces Round 889 (Div. 2)

жңҖе–ңж¬ўиҝҷз§Қд»Җд№Ҳз®—жі•йғҪжІЎжңүзҡ„жһ„йҖ йўҳгҖӮ

з®ҖжҳҺйўҳж„Ҹ

з»ҷе®ҡ nnn дёӘж•° aia_iaiвҖӢпјҢдҪ еҸҜд»ҘиҝӣиЎҢдёҚи¶…иҝҮ kkk ж¬Ўж“ҚдҪңпјҢдҪҝеҫ—жңҖз»Ҳ aaa жҳҜйқһйҷҚеәҸзҡ„гҖӮжҜҸж¬Ўж“ҚдҪңд»»йҖүдёӨдёӘж•°дҪҝеҫ— ai+=aja_i += a_jaiвҖӢ+=ajвҖӢгҖӮ

Easy Versionпјҡ1вүӨnвүӨ201 \le n \le 201вүӨnвүӨ20пјҢвҲ’20вүӨaiвүӨ20-20 \le a_i \le 20вҲ’20вүӨaiвҖӢвүӨ20пјҢk=50k = 50k=50гҖӮ

Hard Versionпјҡ1вүӨnвүӨ201 \le n \le 201вүӨnвүӨ20пјҢвҲ’20вүӨaiвүӨ20-20 \le a_i \le 20вҲ’20вүӨaiвҖӢвүӨ20пјҢk=31k = 31k=31гҖӮ

и§ЈйўҳжҖқи·Ҝ

Easy Version зҡ„и§Јжі•еҫҲеӨҡе°ұдёҚд»Ӣз»ҚдәҶпјҢжҲ‘зӣҙжҺҘеҮәзҡ„ Hard Version зҡ„жҖқи·ҜгҖӮ

йҰ–е…ҲжҲ‘д»¬иҖғиҷ‘дҪҝз”Ё еүҚзјҖе’Ң ж“ҚдҪңпјҢеӣ дёәдёҚдҪҝз”ЁеүҚзјҖе’ҢдјҡеҸ‘зҺ°й—®йўҳеҸҳеҫ—йқһеёёжЈҳжүӢпјҢжҲ‘д»¬еҫҲйҡҫи®Ёи®әеәҸеҲ—д№Ӣй—ҙзҡ„еӨҚжқӮзҠ¶жҖҒгҖӮеҰӮжһңжңҖеҗҺдёҖжӯҘдҪҝз”ЁеүҚзјҖе’ҢпјҢжҲ‘д»¬еҸӘйңҖиҰҒдҝқиҜҒеңЁжңҖеҗҺдёҖжӯҘд№ӢеүҚжүҖжңүж•°йғҪжҳҜйқһиҙҹж•°жҲ–иҖ…йқһжӯЈж•°пјҲеҜ№еә”еҗҺзјҖе’ҢпјүгҖӮеүҚзјҖе’Ңзҡ„д»Јд»·жҳҜ nвҲ’1n-1nвҲ’1гҖӮ

е…¶ж¬ЎжҲ‘д»¬е…Ҳ иҖғиҷ‘е…ЁйғЁеҸҳдёәйқһиҙҹж•°зҡ„ж–№ејҸ пјҢеӣ дёәе…ЁйғЁеҸҳдёәйқһиҙҹе’Ңе…ЁйғЁеҸҳдёәйқһжӯЈжҳҜзӣёеҸҚзҡ„ж“ҚдҪңпјҢжүҖд»Ҙе…ҲиҖғиҷ‘дёҖиҫ№еҚіеҸҜгҖӮ

йӮЈд№Ҳй—®йўҳиҪ¬еҢ–дёәеҰӮдҪ• еңЁеү©дҪҷд»Јд»·еҶ…е°ҶжүҖжңүж•°иҪ¬еҢ–дёәйқһиҙҹж•° гҖӮеҒҮи®ҫжҲ‘д»¬еҜ№дәҺеәҸеҲ— aaa жұӮеҮәжңҖеӨ§жӯЈж•°зҡ„з»қеҜ№еҖј max+max_+max+вҖӢпјҢжӯЈж•°зҡ„дёӘж•° cnt+cnt_+cnt+вҖӢпјҢжңҖеӨ§иҙҹж•°зҡ„з»қеҜ№еҖј maxвҲ’max_-maxвҲ’вҖӢпјҢиҙҹж•°зҡ„дёӘж•° cntвҲ’cnt_-cntвҲ’вҖӢгҖӮйӮЈд№ҲеҫҲзӣҙи§Ӯзҡ„жғіжі•е°ұжҳҜи®©жүҖжңүиҙҹж•°еҠ дёҠ max+max_+max+вҖӢ зӣҙеҲ°еҸҳдёәйқһиҙҹж•°пјҢдҪҶжҳҜеҪ“ max+=1max_+=1max+вҖӢ=1 иҖҢ maxвҲ’=20max_-=20maxвҲ’вҖӢ=20 ж—¶пјҢиҝҷж ·еҒҡзҡ„д»Јд»·жҳҜ 202020гҖӮиҝҷйҮҢйңҖиҰҒи§ӮеҜҹеҲ° aia_iaiвҖӢ жҳҜеҸҜд»ҘеҒҡиҮӘеҠ зҡ„пјҢиҝҷзӣёеҪ“дәҺдёҖдёӘеҖҚеўһзҡ„иҝҮзЁӢпјҢйӮЈд№ҲйҰ–е…Ҳи®© max+max_+max+вҖӢ еҖҚеўһеҲ°еӨ§дәҺзӯүдәҺ maxвҲ’max_-maxвҲ’вҖӢпјҢеҶҚиҝӣиЎҢеҗҺйқўзҡ„ж“ҚдҪңгҖӮи®Ўз®—жҖ»д»Јд»·дёә nвҲ’1+вҢҲlogвҒЎ2maxвҲ’max+вҢү+cntвҲ’n-1 + \lceil\log_2{\frac{max_-}{max_+}}\rceil + cnt_-nвҲ’1+вҢҲlog2вҖӢmax+вҖӢmaxвҲ’вҖӢвҖӢвҢү+cntвҲ’вҖӢ пјҢиҝҷж ·еҒҡзҡ„жңҖеӨ§д»Јд»·дёә 434343пјҢжҳҫз„¶й—®йўҳиҝҳжІЎжңүз»“жқҹгҖӮ

жҲ‘д»¬д№ӢеүҚи®Ёи®әзҡ„жҳҜиҖғиҷ‘е°Ҷе…ЁйғЁж•°еҸҳдёәйқһиҙҹж•°пјҢйӮЈд№Ҳзӣёеә”зҡ„пјҢжҲ‘д»¬иҝҳеҸҜд»ҘйҖүжӢ© е°Ҷе…ЁйғЁеҸҳдёәйқһжӯЈж•°зҡ„ж–№ејҸ пјҢз„¶еҗҺеҒҡдёҖж¬ЎеҗҺзјҖе’ҢгҖӮжҖ»д»Јд»·дёә nвҲ’1+вҢҲlogвҒЎ2max+maxвҲ’вҢү+cnt+n-1 + \lceil\log_2{\frac{max_+}{max_-}}\rceil + cnt_+nвҲ’1+вҢҲlog2вҖӢmaxвҲ’вҖӢmax+вҖӢвҖӢвҢү+cnt+вҖӢгҖӮиҝҷе…¶е®һд№ҹжҳҜдёҖдёӘеҫҲзӣҙи§Ӯзҡ„жғіжі•пјҢдҪҶжҳҜжҲ‘д»¬еҰӮдҪ•еҜ№иҝҷдёӨз§Қж–№жЎҲеҒҡеҮәеҶізӯ–е‘ўпјҹеҰӮжһңеҚЎеңЁиҝҷйҮҢйӮЈе®һеңЁжҳҜеӨӘеҸҜжғңдәҶпјҢдәӢе®һдёҠжҲ‘д»¬еҸҜд»Ҙе°ҶдёӨдёӘж–№жЎҲйғҪиҝӣиЎҢи®Ўз®—пјҢе°ҶеҶізӯ–дәӨз»ҷзЁӢеәҸиҝӣиЎҢпјҲдёҖдёӘз®ҖеҚ•зҡ„ ifвҲ’elseif-elseifвҲ’else иҜӯеҸҘиҖҢе·ІпјүгҖӮжӢҝеҲ°иҝҷдёӘж–№жЎҲжҲ‘д№ҹе№¶дёҚзЎ®е®ҡжҳҜеҗҰз¬ҰеҗҲйўҳзӣ®жқЎд»¶пјҢдҪҶжҳҜеҗҺйқўжҲ‘д»¬е°Ҷ иҜҒжҳҺ гҖӮ

жңҖеҗҺиҝҳеү©дёӢдёҖдәӣдёҠйқўзҡ„и§Јжі•еҸҜиғҪж— жі•и§ЈеҶізҡ„дёҖдәӣе°Ҹй—®йўҳйңҖиҰҒи§ЈеҶіпјҡеҰӮжһңе…Ёдёә 000пјҢдёҚйңҖиҰҒиҝӣиЎҢд»»дҪ•ж“ҚдҪңпјӣеҰӮжһңеҸӘжңүйқһиҙҹж•°пјҢзӣҙжҺҘиҝӣиЎҢдёҖж¬ЎеүҚзјҖе’ҢпјӣеҰӮжһңеҸӘжңүйқһжӯЈж•°пјҢзӣҙжҺҘиҝӣиЎҢдёҖж¬ЎеҗҺзјҖе’ҢгҖӮ

Markdown зі»еҲ—еүҚиЁҖ

еҸ‘иЎЁдәҺ2023-07-28|жӣҙж–°дәҺ2023-07-28|Markdown
|еӯ—ж•°жҖ»и®Ў:391|йҳ…иҜ»ж—¶й•ҝ:1еҲҶй’ҹ|йҳ…иҜ»йҮҸ:

еүҚиЁҖ

Markdown жҳҜдёҖз§ҚиҪ»йҮҸзә§ ж Үи®°иҜӯиЁҖпјҢе®ғд»ҘзәҜж–Үжң¬еҪўејҸпјҲжҳ“иҜ»гҖҒжҳ“еҶҷгҖҒжҳ“жӣҙж”№пјүзј–еҶҷж–ҮжЎЈпјҢе№¶жңҖз»Ҳд»Ҙ HTML ж јејҸеҸ‘еёғгҖӮMarkdown д№ҹеҸҜд»ҘзҗҶи§Јдёәе°Ҷд»Ҙ Markdown иҜӯжі•зј–еҶҷзҡ„иҜӯиЁҖиҪ¬жҚўжҲҗ HTML еҶ…е®№зҡ„е·Ҙе…·гҖӮ

йҖүжӢ© Markdown жҳҜеӣ дёә Markdown еҸҜд»Ҙи®ӨдёәжҳҜз®ҖеҢ–зүҲзҡ„ HTMLпјҢдҪҝз”Ё HTML еҸҜд»Ҙзј–еҶҷдјҳзҫҺзҡ„ж–ҮжЎЈпјҢдҪҶ HTML еӨ§йҮҸзҡ„ж Үзӯҫе№¶дёҚжҳҜеҜ№дәҺд№ҰеҶҷеҸӢеҘҪгҖӮMarkdown еңЁжңҖз®ҖеҢ–з”ЁжҲ·д№ҰеҶҷзҡ„жғ…еҶөдёӢдёәз”ЁжҲ·жҸҗдҫӣдәҶиҫғдёәеӨҡж ·зҡ„ж–Үжң¬ж јејҸгҖӮеҪ“з„¶пјҢеҰӮжһңйңҖиҰҒпјҢеӨ§еӨҡж•° Markdown зј–иҫ‘еҷЁйғҪж”ҜжҢҒзӣҙжҺҘдҪҝз”Ё HTML иҜӯжі•гҖӮ

Markdown зј–иҫ‘еҷЁз§Қзұ»з№ҒеӨҡпјҢе№¶жІЎжңүеҚҒеҲҶз»ҹдёҖзҡ„ж ҮеҮҶпјҢдёҚеҗҢзј–иҫ‘еҷЁзҡ„жёІжҹ“ж•Ҳжһңд№ҹе·®ејӮжһҒеӨ§гҖӮеҗҢж—¶дёҚеҗҢзј–иҫ‘еҷЁж”ҜжҢҒеҗ„дёҚзӣёеҗҢзҡ„жӢ“еұ•пјҢеҗҺйқўзҡ„ж–Үз« е°Ҷдјҡд»Ӣз»Қ Markdown еҺҹз”ҹиҜӯжі•гҖҒLATEX ж•°еӯҰе…¬ејҸгҖҒMermaid з»ҳеӣҫгҖҒHexo ж ҮзӯҫеӨ–жҢӮгҖӮе…¶дёӯпјҢMarkdown еҺҹз”ҹиҜӯжі•дёҚеҗҢзј–иҫ‘еҷЁеҹәжң¬зӣёеҗҢпјӣLATEX ж•°еӯҰе…¬ејҸд№ҹжҳҜеӨҡж•°зј–иҫ‘еҷЁж”ҜжҢҒзҡ„пјҢдҪҶж”ҜжҢҒдёҠеҸҲжңүз•Ҙеҫ®е·®ејӮпјӣMermaid з»ҳеӣҫжҳҜдёҖдәӣй«ҳзә§зј–иҫ‘еҷЁжүҚжӢҘжңүзҡ„ж”ҜжҢҒпјӣHexo ж ҮзӯҫеӨ–жҢӮд»…еңЁ Hexo дёӯиў«ж”ҜжҢҒпјҢHexo жҳҜиҝҷдёӘеҚҡе®ўзҡ„жЎҶжһ¶пјҢеҰӮжһңдҪ дёҚйңҖиҰҒеӯҰд№ дҪҝз”Ё Hexo иҜ·и·іиҝҮиҝҷдёӘз« иҠӮгҖӮ

еңЁејҖе§Ӣд№ӢеүҚиҜ·дёӢиҪҪеҘҪдҪ зҡ„ Markdown зј–иҫ‘еҷЁпјҢжҺЁиҚҗдҪҝз”ЁпјҡTypora вҖ” a markdown editor, markdown reader.

Github + Hexo + Butterfly е»әз«ҷз¬”и®°(еӣӣ) - PicGo еӣҫеәҠй…ҚзҪ®

еҸ‘иЎЁдәҺ2023-07-26|жӣҙж–°дәҺ2023-07-26|е»әз«ҷз¬”и®°
|еӯ—ж•°жҖ»и®Ў:531|йҳ…иҜ»ж—¶й•ҝ:1еҲҶй’ҹ|йҳ…иҜ»йҮҸ:

дёәд»Җд№ҲжҲ‘д»¬йңҖиҰҒеӣҫеәҠпјҹ

еӣ дёәжҲ‘д»¬зҡ„еҚҡе®ўдёҠдј еҲ°дә‘з«ҜеҗҺпјҢдјҡдёўеӨұе…¶дёӯеӣҫзүҮеңЁжң¬ең°зҡ„й“ҫжҺҘгҖӮеӣҫеәҠжҳҜдёҖдёӘеңЁдә‘з«ҜеӯҳеӮЁеӣҫзүҮзҡ„е№іеҸ°пјҢдҪҝз”ЁеӣҫеәҠеҸҜд»ҘжҜ”иҫғеҘҪзҡ„и§ЈеҶіиҝҷдёӘй—®йўҳгҖӮ

иҝҷйҮҢжҲ‘д»¬е°ҶдҪҝз”Ё PicGo + Github е®һзҺ°еӣҫеәҠпјҢе…¶дёӯ PicGo еҸӘжҳҜдёҖдёӘжң¬ең°зҡ„еӣҫеәҠз®ЎзҗҶе·Ҙе…·пјҢиҖҢ Github дёәе…¶жҸҗдҫӣеӯҳеӮЁгҖӮ

PicGo е®ҳж–№ж–ҮжЎЈ

1. еҲӣе»ә Github Tokens

  • йҰ–е…ҲеңЁ Github зҡ„дёӘдәәи®ҫзҪ®жңҖдёӢж–№жүҫеҲ° Developer Settings гҖӮ

  • йҖүжӢ© Personal access tokens дёӢзҡ„ Tokens (classic) йҖүйЎ№пјҢз„¶еҗҺ Generate new token гҖӮ

  • еңЁ Note дёӯдёәиҝҷжқЎ token еӨҮжіЁпјҢе№¶дё”и®ҫзҪ® Expiration еҲ°жңҹж—¶й—ҙпјҢеҸҜд»Ҙи®ҫзҪ®дёәж— йҷҗжңҹпјҢз„¶еҗҺиҝҷжқЎ token йңҖиҰҒејҖеҗҜжүҖжңү repo жқғйҷҗгҖӮеңЁеҲӣе»ә token еҗҺиҜҘ token еҸӘдјҡжҳҫзӨәдёҖж¬ЎпјҢиҜ·еҰҘе–„дҝқеӯҳпјҢеҗҺйқўе°Ҷдјҡз”ЁеҲ°иҝҷжқЎ token гҖӮ
  • token еҰӮжһңиў«еҲ«дәәиҺ·еҸ–пјҢе®ғеҸҜд»ҘдҪҝз”ЁиҜҘ token е…·жңүзҡ„жқғйҷҗж“ҚдҪңдҪ зҡ„иҙҰжҲ·пјҢе°Ҷ token и®ҫзҪ®дёәж— йҷҗжңҹжҳҜеҚұйҷ©зҡ„пјҢе№¶дё”дёҚиҰҒе°ҶдҪ зҡ„ token д»»ж„Ҹдј ж’ӯгҖӮеҰӮжһңеҸ‘зҺ° token дёўеӨұжҲ–жҖҖз–‘ token жі„йңІпјҢиҜ·еҠЎеҝ…з«ӢеҚіеҲ йҷӨеҺҹ token е№¶еҲӣе»әж–°зҡ„ token йҮҚж–°й…ҚзҪ®зӣёе…іеә”з”ЁгҖӮ

2. дёӢиҪҪ PicGo

  • еүҚеҫҖ PicGo е®ҳзҪ‘пјҢжүҫеҲ°жңҖиҝ‘зҡ„зЁіе®ҡзүҲжң¬пјҢеҰӮжһңжҳҜ Windows зі»з»ҹиҜ·дёӢиҪҪ PicGo-Setup-xxx-ia32.exe е№¶жҢүз…§жҢҮзӨәе®үиЈ…е№¶иҝҗиЎҢгҖӮ

  • еҗҜеҠЁ PicGo иІҢдјјдёҚдјҡзӣҙжҺҘеј№зӘ—пјҢе®ғеҸҜиғҪдјҡжңҖе°ҸеҢ–еҲ°дҪ зҡ„жүҳзӣҳдёӯпјҢеҸҜд»ҘеңЁжүҳзӣҳдёӯжү“ејҖгҖӮ

3. й…ҚзҪ® PicGo

  • иҝҷйҮҢжҲ‘йҖүжӢ©дҪҝз”Ё Github еҪ“дҪңеӯҳеӮЁд»“еә“пјҢPicGo зҡ„з•ҢйқўйқһеёёеҸӢеҘҪпјҢдҪ д№ҹеҸҜд»ҘжҢүз…§е–ңеҘҪй…ҚзҪ®е…¶д»–е№іеҸ°гҖӮ
  • еҪ“з„¶пјҢйҰ–е…ҲйңҖиҰҒдҪ еңЁ Github ж–°е»әд»“еә“пјҢдҪ зҡ„жүҖжңүеӣҫзүҮйғҪдјҡеӯҳеӮЁеҲ°иҜҘд»“еә“дёӯпјҢжҲ‘е°Ҷд»“еә“е‘ҪеҗҚдёә PicBed гҖӮ

  • и®ҫзҪ®д»“еә“еҗҚдёӯиҜ·дҪҝз”Ё з”ЁжҲ·еҗҚ/д»“еә“еҗҚ зҡ„ж јејҸпјҢеңЁ Token дёӯзІҳиҙҙеҲҡеҲҡеҲӣе»әзҡ„ token пјҢеӯҳеӮЁи·Ҝеҫ„жҳҜд»“еә“дёӯеӯҳеӮЁеӣҫзүҮзҡ„дҪҚзҪ®гҖӮ

  • зӣҙжҺҘе°ҶеӣҫзүҮжӢ–жӢҪеҲ°жЎҶдёӯеҚіеҸҜе®ҢжҲҗеӣҫзүҮзҡ„дёҠдј гҖӮеҸідёҠи§’еҠ еҸ·еҲҷжҳҜжӮ¬жө®зӘ—еҠҹиғҪпјҢеҗҢж ·ж”ҜжҢҒжӢ–жӢҪдёҠдј гҖӮ
